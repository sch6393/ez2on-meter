EZ2ON METER
===

### 설명
* EZ2ON 판정 미터기에서 COOL 판정과 GOOD 판정을 따로 빼 게임 중에 쉽게 확인하기 위함

<br>

### 요구사항
1. [EZ2ON REBOOT R](https://store.steampowered.com/app/1477590/EZ2ON_REBOOT__R/)
1. .NET Framework v4.8

<br>

### 문제점
- [ ] 게임 기어가 가운데 있을 때만 가능
- [ ] 해상도에 따라 좌표가 살짝 어긋나는 현상이 있음
